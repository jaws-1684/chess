## Installation

Run the following command to install this game as gem
As a gem:

```sh
	gem install chess-1.1.0.gem
```
Run game the from your terminal `chess`
In case you dont want to install the gem, first install the dependecies:
```sh
	bundle install 
```
And run the game with `bin/chess`
